  1. Поделитесь статьёй в используемой Вами социальной сети, используя виджет социальных кнопок на боковой стороне этой страницы.

  2. Подпишитесь на все статьи:

    {% include 'includes/addtoany-rss.html' %}

    Или на конкретные теги/категории через предпочитаемый Вами RSS-агрегатор. Я использую QuiteRSS для Windows.

Проблемы? ¶

  1. (Только для ПК, не мобильных устройств). Eсли Вы увидели на данной странице опечатку или мелкую ошибку, пожалуйста, сообщите об этом, используя typo-reporter. Выделите текст с опечаткой/ошибкой → нажмите Ctrl+Enter (⌘Enter для Mac) → напишите комментарий → Отправить.
  2. Если Вы нашли более серьёзные недочёты для этой страницы или сайта в целом, создайте issue на GitHub.
  3. Также Вы можете самостоятельно редактировать эту страницу и отправить pull request.
  4. Или написать комментарий здесь. Для комментариев поддерживается синтаксис Markdown GFM. Через GhostText Вы можете набирать сообщение в своём любимом Markdown-редакторе.
